## Features
Shaders<br>
Textures<br>
Mesh<br>
Framebuffer (Vignette)<br>
Build in ImGui<br>
Build in window using GLFW<br>

## Dependencies (Arch Linux)

```
sudo pacman -S glfw
sudo pacman -S glm
```

## How to run it?
simply create build folder and run<br>
```cmake ..```<br>
then you can run the app using<br>
```make run```<br>
